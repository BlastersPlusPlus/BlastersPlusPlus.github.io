This tool lets you adapt your old save file to work with the latest version of Blasters ++. To use it, simply extract the file from your game and upload them above. If you don't know how to get the save files, follow the steps below.

3DS

You will need Checkpoint installed on your system. You can install it from this link by scanning the QR code or by downloading and installing the cia or 3dsx file. On your console, open Checkpoint and navigate to your title. Press A once to select it, then select "New...". Press A to confirm, then input a name for the save backup. Your save will be in your SD card, on 3ds/Checkpoint/saves/<id> <name>/<your backup name>, where <id> is the last five characters of the title id.

To load the converted files back, duplicate the folder with the save backup's name and rename it to whatever you'd like. This will ensure that if something goes wrong with the conversion, your data will not be lost. Put the edited files in this new folder. Insert your SD card back into your console, open Checkpoint, select your title, and navigate to the new backup that should appear in the bottom screen. Press A to restore the backup, and A again to confirm.

Citra

Right-click on your title and select 'Open Save Data Location'. Your file explorer should open at a directory with the save files. We recommend storing a copy of them somewhere safe in case something goes wrong during the conversion.

To load the converted files, simply put them in the same directory you got them from, overwriting as necessary.

On both methods, there will be up to six save files, one for each save slot you have used in-game. They are named game<number>.yw(_g). The number matches the diary slot the file corresponds to, whereas the extension is .yw for Red Cat Corps and White Dog Squad files and .yw_g for Moon Rabbit Crew files. We recommend converting all of them

If you need any help using the tool, you can ask in our Discord server's help forum. Likewise, if something goes wrong with it, please report it in the server's bug report forum.
