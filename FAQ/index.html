How do I install Blasters ++?

3DS

You will need a modded console with Custom Firmware (CFW) for this. For information on 3DS CFW and how to install it or know if you already have it, visit 3ds.hacks.guide

  1. Download the mod file and access your 3DS's SD Card. This can be done by placing it into your PC, or, on New 3DS consoles, accessing it via the Internet through System Settings → Data Management.
  2. Navigate through your 3DS's SD Card by going to 'luma/titles.' In short, you should be inside the titles folder, with your full folder path being '[SD Card]/luma/titles.'
  3. Locate your YO-KAI WATCH BLASTERS titleID. This can vary depending on the version and region you have. If the titleID folder you need does not exist, create one with the titleID you need. Here are the titleIDs for the international releases of the game:
    • RCC PAL: 00040000001CEC00
    • RCC NA: 00040000001CEB00
    • WDS PAL: 00040000001CF000
    • WDS NA: 00040000001CEF00
    Once you've located or created your titleID folder, go inside this folder.
  4. Open the mod file that you downloaded earlier using an appropiate program such as WinRAR. Inside it there will be a 'romfs' folder. Drag this folder into the folder you're in.
  5. Power off your console. With the SD card inserted, hold SELECT while powering it on. This should open the Luma menu. Make sure 'Enable game patching' is set (marked as (x)). If it's not, enable it and save the changes.

Citra

  1. Download the mod file.
  2. Open Citra, and right-click on your copy of YO-KAI WATCH BLASTERS. Select the option that says 'Open Mods Location.'
  3. Open the mod file that you downloaded earlier with an appropiate program. Inside it there will be a 'romfs' folder. Drag this folder into the folder that Citra has just opened.
  4. Make sure you don't have any mods in your mods folder.

If successful, when starting the game, the Level5 intro will be replaced by a custom one.

What game versions does Blasters ++ support?

Blasters ++ is supported in the NA (North America) and PAL (rest of the world, excluding Korea and Japan) versions of the game. The mod is only supported in English —playing in any other language will result in issues, including game crashes. To play Blasters ++, you must be using the latest update available for your region.

Blasters ++ isn't working, how do I fix it?

Make sure the following are true for your platform.

3DS

  1. Your console is modded and has CFW.
  2. Game Patching is enabled in Luma Settings (access the Luma menu by holding SELECT while powering on the console).
  3. You are using the correct title ID for your game (e.g. use title ID 00040000001CEB00 for Red Cat Corps North American version).
  4. Your file path looks as follows: root/luma/titles/<TITLE_ID>/romfs/<MOD_FILES>
  5. You are playing on the latest update.

Citra

  1. You placed the "romfs" folder into the game's mods folder

If the above doesn't work, you can ask for support in our Discord server.

How do I update Blasters ++?

To update the mod simply drag the romfs folder from the latest update into the title ID folder and replace any existing files

If I turn off Blasters ++, will I lose my Yo-kai?

Yes, you will lose your Blasters ++ Yo-kai. Without the mod active, the game can’t identify the modded Yo-kai, so it deletes them. The game only deletes the Blasters ++ Yo-kai when you boot the game and save, so if game patching is disabled but the game wasn’t saved the Yo-kai should be fine.

Additionally, if the Yo-kai are deleted from the game by turning off the mod and saving, the game will still register the medal as taking up a medal slot, meaning a medal slot will be lost forever.

The same applies to other content introduced by the mod, such as new items. We recommend keeping two separate save files if you plan to alternate between using and not using Blasters ++.

Can I play Blasters ++ online?

3DS

The 3DS online servers shut down in April of 2024, but there are custom-built servers for several games. For Blasters, there is Level256 network, which disables game patching, meaning that you cannot regularly use Blasters ++ and Level256 at the same.
At the time of writing, Pretendo Network does not support Blasters yet.

Citra

You can play online on Citra by changing some configurations to enable the use of different lobbies

In order to do this, you need to open the qt-config.ini Citra configuration file. On Windows, this file is found at C:\Users\<your username>\AppData\Roaming\Citra\config. Open it using Notepad and search for WebService (you can do this with the shortcut Ctrl+F if your device is in English). Replace the web_api_url lines with the following:

web_api_url\default=false
web_api_url=http:/88.198.47.46:5000

After saving your changes (and ensuring the save was succesful!), you should be able to join multiplayer rooms. If Citra is currently open, you will need to close and reopen it to let the changes take place.

To play with others, you will need to join a room. In the multiplayer menu, you can either browse public rooms or directly join one given its address and port. Once in the room, you can open the game and go to local multiplayer to start playing.

Note that, when playing with others without the mod, they won't be able to see Blasters ++ Yo-kai. Depending on who hosts the room, other behaviors will also change.
When playing Blasters ++ online, make sure to avoid playing with users without the mod (or with different versions of it) so as to not ruin their experience. Likewise, follow the guidelines of whatever online service you are using.

Why can't I see the new Yo-kai in the Medallium?

This game has some internal limitations regarding things like the Medallium. In particular, Medallium numbers over 511 cause issues, including crashes. Due to this, we are not able to add any more entries. However, you can find all the information you'd see in the Medallium (including the Medallium Bio) and more at Yo-kai Info!

I don't have access to Jibanyan B and Komasan B after beating Whisped Cream

This is normal. The mod changes the unlock condition of both to be beating Captain Thunder's story mission, the same as Usapyon B. This is because it's the only way to make them unlock for both versions without overcomplicating the installation process.

Blasters ++ doesn't work with Level256 or after installing it

Your console has the game update (Moon Rabbit Crew) installed which has several resources within it that make the game work. In the mod romfs folder there are additional and modified files that contain the new content added by Blasters ++. The usual method to play the mod relies on the console loading these external assets when loading the game, and applying them over the existing files within the update installed on the console, but the boot.firm required to use the Level256 plugin disables this. To use this method again, you need to replace the Level256 boot.firm with the one from Luma, which you can find inside Luma3DS.zip from the Github repository.
